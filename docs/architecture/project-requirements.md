# CloudDrive - Project Requirements

## Project Name

CloudDrive

---

## Project Description

CloudDrive is a cloud-based file storage and management platform inspired by Google Drive and Dropbox.

The application allows users to securely upload, organize, share, and manage files from anywhere.

The project focuses on learning Full Stack Development, Cloud Computing, and DevOps practices.

- Visit landing page
- Register
- Login

---

## 2. User

Can:

- Upload files
- Download files
- Delete files
- Create folders
- Rename folders
- Search files
- Share files
- View storage usage
- Update profile

---

## 3. Admin

Can:

- View all users
- Delete users
- View storage statistics
- Monitor platform activity
